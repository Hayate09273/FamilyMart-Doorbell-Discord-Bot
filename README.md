# FamilyMart-Doorbell-Discord-Bot
DiscordのVCに入室するとファミリーマートの入店音を流します。

## 必要なもの
+ Python
+ discord.py[voice]
+ FFmpeg
+ Discord BOT

## コマンド
|コマンド|動作|
|:-:|:-:|
|;join|BotをVCに呼び出します。|
|;leave|BotをVCから退出させます。|
